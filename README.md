# OhMyFooD 

Projet d'école pour les animations CSS 

## Dependances 

Le projet utilise npm, pensez à lancer la commande `npm install` pour récupérer la dépendance **SASS**
Pour lancer le projet en developpement, utilisez la commande : 
```
npm run sass
```
